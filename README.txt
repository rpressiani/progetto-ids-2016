-To start the game you need, in order, to launch the class "server" in server package,
 then launch the class "client" in client package.

-Default ip address of our server is 127.0.0.1: need to select this address to connect to server

-There is a timeout of 40 seconds for joining a starting match (from 2 to N players)
 and there is the same timeout for each player to make succesfully a move, or you will be disconnected

-When playing by gui, you need to resize the window to show all buttons
